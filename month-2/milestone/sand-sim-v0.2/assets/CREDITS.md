# Credits

Attribution for third-party assets used in `sand-sim` v0.2 (audio added in Session 16).

> This file is a placeholder. It'll be populated when the Session 16 audio effects are added — typically CC0 (public domain) WAVs from [freesound.org](https://freesound.org) or [opengameart.org](https://opengameart.org). Each entry will list source URL, author, licence, and any modifications.

## Format

When you add a sound, append an entry like:

```
### fire_crackle.wav
- Source: https://freesound.org/people/<author>/sounds/<id>/
- Author: <name>
- Licence: CC0 (public domain)
- Modifications: trimmed to 1.2s, normalised to -6 dBFS
```

Repeat for each WAV.
